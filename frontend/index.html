Sobre nós

Olá, seja bem-vindo ao site da empresa XyzCréditos. Nossa empresa consiste em um software para que, pessoas possam registrar informações de devedores.

Uma vantagem de usar esse software é que, além de o software ser totalmente benevolente, não é desorganizado, pelo contrário, é organizado.

Ou seja, ao registrar essas informações, você obviamente poderá consultá-la, mas para que possa fazer isso, é necessário ter um cadastro. Ressaltando outras características desse software é que, essas informações ficarão disponíveis para todos usuários com acesso ao sistema.

Você poderá adicionar uma dívida, remover, e se adicionar algum número acidentalmente não será necessário remover o valor. Você pode editá-lo.
